## 📅 日常交易流程

### 开盘前准备（9:00 AM）

**1. 检查持仓状态**
```
Portfolio页面 → 点击"🔄 Refresh Prices"
```
- 查看所有活跃持仓的最新价格
- 检查是否有红色警告（止损触发、亏损过大）
- 更新Current Score（如果超过2天未更新）

**2. 查看到期提醒**
- 黄色警告"7d to exp!" → 考虑提前平仓或展期
- 红色警告"⚠️ HIT STOP" → 立即平仓

**3. 制定当日计划**
- 哪些持仓需要止盈？
- 哪些持仓需要止损？
- 是否有新的入场机会？

---

### 盘中交易（9:30 AM - 4:00 PM）

**场景1: 发现新机会**

```
1. Scanner页面
   ↓
2. 输入Ticker（如QQQ）+ 选择策略（Long）
   ↓
3. 点击"Scan Options"
   ↓
4. 查看评分结果（80+分 = 优秀）
   ↓
5. 点击"Add to Watchlist"
   ↓
6. 填写理想入场价、止损条件
   ↓
7. 等待价格到达理想区间
```

**场景2: 执行入场**

```
1. Watchlist页面
   ↓
2. 找到目标持仓
   ↓
3. 确认当前价格 ≈ 理想入场价
   ↓
4. 点击"Move to Active"
   ↓
5. 在弹窗中输入买入数量和实际价格
   ↓
6. 确认 → 持仓转为Active状态
```

**场景3: 加仓/减仓**

```
1. Portfolio页面
   ↓
2. 找到目标持仓卡片
   ↓
3. 点击"Actions" → 选择"Size Up"或"Size Down"
   ↓
4. 输入数量和价格
   ↓
5. 确认 → 平均成本自动更新
```

**场景4: 止盈/止损**

```
1. Portfolio页面
   ↓
2. 看到红色警告"⚠️ HIT STOP"
   ↓
3. 点击"Actions" → "Close"
   ↓
4. 输入平仓数量和价格
   ↓
5. 确认 → 持仓移至History
```

---

### 收盘后复盘（4:30 PM）

**1. 更新所有Score**
```
Portfolio页面 → 逐个点击"Update Score"
```
- 输入最新的Scanner Score
- 系统自动记录更新时间

**2. 查看统计数据**
```
Stats页面 → 查看绩效指标
```
- 总P&L
- 胜率
- 平均持仓时间
- 最佳/最差交易

**3. 记录交易笔记**
```
点击持仓卡片 → 编辑Notes
```
- 为什么入场？
- 为什么出场？
- 学到了什么？

## 🔍 Scanner使用指南

### 基本使用

**1. 输入参数**
- **Ticker**: 股票代码（如QQQ、SPY、AAPL）
- **Strategy**: 
  - `Long` - 买入期权（看涨/看跌）
  - `Short` - 卖出期权（收取权利金）

**2. 高级过滤器（可选）**
- **DTE Range**: 到期天数范围（默认20-60天）
- **Strike Range**: 行权价范围（默认±30%）
- **Min Volume**: 最小成交量（默认50）
- **Max Spread**: 最大买卖价差（默认10%）

**3. 点击"Scan Options"**
- 系统自动获取完整期权链
- 计算所有指标（Lambda、Gamma Eff、Theta Burn）
- 应用过滤器
- 计算最终得分

**4. 查看结果**
- 按得分排序（高到低）
- 绿色 = 80+分（优秀）
- 蓝色 = 70-79分（良好）
- 灰色 = 60-69分（一般）
- 红色 = <60分（不建议）

---

### 评分解读

**LOQ Score (买方)**

**80-100分 - 优秀机会**
- ✅ 高杠杆（Lambda 5-15）
- ✅ 高爆发力（Gamma Eff 0.01-0.05）
- ✅ 低时间衰减（Theta Burn <0.02）
- ✅ ATM区间（Delta 0.30-0.50）
- ✅ IV便宜（Contango）

**示例**: QQQ $630 Call, 45 DTE, Delta 0.45, Score 85
- 理想的swing trade机会
- 适合持有1-3周

**70-79分 - 良好机会**
- ✅ 大部分指标优秀
- ⚠️ 某些指标略差（如IV偏贵）

**示例**: SPY $580 Put, 30 DTE, Delta 0.35, Score 72
- 可以考虑，但需谨慎
- 适合短期交易（1周内）

**60-69分 - 一般机会**
- ⚠️ 多个指标不理想
- 🚫 不建议新手交易

**<60分 - 不建议**
- 🚫 彩票式期权（Delta <0.15）
- 🚫 IV过贵（Backwardation）
- 🚫 时间衰减过快

---

### IV Term Structure解读

**Contango (IV Ratio < 0.95)**
- ✅ **适合买入**
- 近期IV低，远期IV高
- 期权相对便宜

**Neutral (IV Ratio 0.95-1.05)**
- ⚠️ **谨慎买入**
- IV正常水平
- 需要其他指标支持

**Backwardation (IV Ratio > 1.05)**
- 🚫 **不建议买入**
- ✅ **适合卖出**
- 近期IV高，市场恐慌
- 期权过贵

## 📝 Watchlist管理

### 添加到Watchlist

**方法1: 从Scanner添加**
```
Scanner → 找到高分期权 → "Add to Watchlist"
```

**方法2: 手动添加**
```
Watchlist → "+" 按钮 → 填写信息
```

### 必填信息

| 字段 | 说明 | 示例 |
|------|------|------|
| Ticker | 股票代码 | QQQ |
| Strike | 行权价 | 630 |
| Type | Call/Put | Call |
| Expiration | 到期日 | 2026-02-20 |
| Entry Score | Scanner评分 | 85 |
| Ideal Entry | 理想入场价 | 7.50 |
| Stop Reason | 止损条件 | MB flips red |

### 监控Watchlist

**每日检查**:
1. 当前价格 vs 理想入场价
2. Score是否仍然>70
3. 距离到期是否太近

**入场时机**:
- ✅ 当前价格 ≤ 理想入场价
- ✅ Score仍然>70
- ✅ 距离到期>20天

**移除条件**:
- ❌ Score降至<60
- ❌ 距离到期<7天
- ❌ 价格远离理想区间

## 💼 持仓管理

### 持仓卡片信息

**基本信息**:
- Ticker, Strike, Type, Expiration
- Contracts（当前持仓数量）
- Entry Price（平均成本）
- Current Price（当前价格）

**P&L显示**:
- 百分比（绿色=盈利，红色=亏损）
- 美元金额

**Score对比**:
- Entry Score（入场时）
- Current Score（当前）
- 趋势箭头（↑↓）

**警告标签**:
- 🔴 Danger（立即行动）
- 🟡 Warning（注意）
- 🔵 Info（提示）

---

### 操作按钮

**🔄 Auto Price**
- 自动获取最新价格和Greeks
- 更新数据库
- 刷新P&L

**✏️ Manual**
- 手动输入价格（API失败时）

**📊 Update Score**
- 手动更新Scanner Score
- 建议每2天更新一次

**⚙️ Actions**
- Size Up（加仓）
- Size Down（减仓）
- Take Profit（部分止盈）
- Close（平仓）

---

### Greeks历史图表

**展开方式**:
```
点击持仓卡片 → "Show Greeks History"
```

**图表内容**:
- **蓝线**: IV（隐含波动率）
- **绿线**: Delta（方向性）
- **X轴**: 时间（最近7天）

**用途**:
- 观察IV趋势（上升=变贵，下降=变便宜）
- 观察Delta变化（接近1=深度ITM，接近0=OTM）

## 🛡️ 风险管理

### 止损规则

**技术止损**
- 在Watchlist阶段设定
- 例: "MB flips red", "Breaks below 620"
- 触发后立即平仓

**百分比止损**
- 默认: -40%触发红色警告
- 建议: -30%时考虑止损
- 严格: -20%时强制止损

**时间止损**
- 距离到期<7天 → 黄色警告
- 距离到期<3天 → 强制平仓

---

### 仓位管理

**单个持仓**:
- 最大: 账户的10%
- 推荐: 账户的5%

**总持仓**:
- 最大: 账户的50%
- 推荐: 账户的30%

**分散化**:
- 不超过3个相同Ticker
- 不超过5个相同Setup
- 不超过10个总持仓

---

### 加仓策略

**金字塔加仓**（推荐）:
```
初始: 3份 @ $7.50
盈利10%: +2份 @ $8.25
盈利20%: +1份 @ $9.00
```

**平均成本加仓**（谨慎）:
```
初始: 3份 @ $7.50
下跌10%: +2份 @ $6.75
下跌20%: +1份 @ $6.00
```
⚠️ 只在Score仍然>70时使用

## ✅ 最佳实践

### 入场纪律

**强制使用Watchlist**
- ❌ 不要冲动交易
- ✅ 所有机会先加入Watchlist
- ✅ 等待价格到达理想区间

**Score要求**
- ✅ 只交易Score>70的机会
- ✅ Score>80优先考虑
- ❌ 绝不交易Score<60

**IV检查**
- ✅ Contango时买入
- ⚠️ Neutral时谨慎
- ❌ Backwardation时避免

---

### 出场纪律

**止盈策略**
- 目标1: +50% → 平仓50%
- 目标2: +100% → 平仓30%
- 目标3: 让利润奔跑

**止损策略**
- 技术止损触发 → 立即平仓
- 亏损-30% → 考虑止损
- Score<60 → 立即平仓

**时间管理**
- 距离到期<7天 → 考虑平仓
- 距离到期<3天 → 强制平仓
- Theta加速期（最后2周）→ 谨慎持有

---

### 记录习惯

**每日记录**
- 更新所有Score
- 刷新所有价格
- 检查警告标签

**每笔交易记录**
- 入场理由（Setup, Score, IV）
- 出场理由（止盈/止损/时间）
- 学到的教训

**每周复盘**
- 查看Stats页面
- 分析胜率和P&L
- 识别成功/失败模式

---

### 常见错误

**❌ 错误1: 追涨杀跌**
- 看到大涨就买入（FOMO）
- 看到大跌就卖出（恐慌）

**✅ 正确做法**:
- 使用Watchlist等待理想价格
- 遵守预设的止损/止盈计划

**❌ 错误2: 不止损**
- 亏损-50%还持有
- 希望"回本"

**✅ 正确做法**:
- 严格执行-30%止损
- 接受小亏损，保护资本

**❌ 错误3: 过度交易**
- 每天都要交易
- 同时持有10+个持仓

**✅ 正确做法**:
- 只交易高质量机会（Score>80）
- 保持3-5个核心持仓

**❌ 错误4: 忽略Greeks**
- 只看价格，不看IV
- 不关注Theta衰减

**✅ 正确做法**:
- 每2天更新Greeks
- 查看Greeks历史图表
- 避免IV过贵的期权

## 🎯 成功案例

### 案例1: 完美的Swing Trade

**Setup**: QQQ Call, MB Bounce

**流程**:
```
Day 1: Scanner发现 (Score 85, IV Ratio 0.92)
       → 添加到Watchlist
       → 理想入场: $7.50

Day 3: 价格到达$7.45
       → Move to Active
       → 买入5份 @ $7.45

Day 7: 价格上涨至$9.20 (+23%)
       → 部分止盈3份 @ $9.20

Day 12: 价格上涨至$11.50 (+54%)
        → 平仓剩余2份 @ $11.50

结果: 总收益 +$1,935 (+52%)
```

**关键点**:
- ✅ 耐心等待理想入场价
- ✅ 分批止盈
- ✅ 在到期前2周平仓

---

### 案例2: 及时止损避免大亏

**Setup**: SPY Put, Breakdown

**流程**:
```
Day 1: Scanner发现 (Score 78, IV Ratio 1.02)
       → 添加到Watchlist
       → 理想入场: $5.20
       → 止损条件: "Breaks above 585"

Day 2: 价格到达$5.15
       → Move to Active
       → 买入4份 @ $5.15

Day 4: SPY突破585（止损触发）
       → 立即平仓 @ $4.50

结果: 亏损 -$260 (-12.6%)
```

**关键点**:
- ✅ 预设止损条件
- ✅ 严格执行止损
- ✅ 小亏损保护资本
